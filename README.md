#### 软件架构

软件架构说明

后端技术：Spring Boot 2.6.6 +MybatisPlus 3.5.1 + Shiro 1.8.0+ Jwt 3.11.0 
          + Swagger + Redis 

前端技术： Ant-design-vue + Vue

GIS框架：mapboxgl + leaflet + Cesium

其他技术： Druid（数据库连接池）、Logback（日志工具） 、poi（Excel工具）、
          Quartz（定时任务）、lombok（简化代码）

项目构建： Maven、Jdk8
